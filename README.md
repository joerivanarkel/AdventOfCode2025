# Advent of Code 2025

My solutions for [Advent of Code 2025](https://adventofcode.com/2025) in F#.

## Getting Started

### Prerequisites

- .NET SDK 10.0 or later
- Visual Studio Code with Ionide extension (recommended)

### Running the Solutions

```bash
dotnet build
dotnet run --project src/AdventOfCode2025/AdventOfCode2025.fsproj
```

### Running Tests

```bash
dotnet test
```

## Project Structure

```
AdventOfCode2025/
├── src/
│   └── AdventOfCode2025/          # Main project with solutions
├── tests/
│   └── AdventOfCode2025.Tests/    # Unit tests
└── .devcontainer/                 # GitHub Codespaces configuration
```

## Development

This repository is configured for GitHub Codespaces. Open in a Codespace to get started with a pre-configured F# development environment.
